# 🚀 RealTask — Современный Канбан-менеджер задач

**RealTask** — это Fullstack-приложение для эффективного управления проектами и задачами. Построено на передовых технологиях для обеспечения скорости, надежности и отличного пользовательского опыта.

## ✨ Основные возможности

- **🔐 Полная авторизация**: Регистрация и вход через NextAuth.js (защищенные роуты и сессии).
- **📋 Интерактивный Канбан**: Drag-and-drop перемещение задач между колонками (To Do, In Progress, Review, Done).
- **📂 Управление проектами**: Создание, редактирование и каскадное удаление проектов (вместе со всеми задачами).
- **💬 Система комментариев**: Обсуждение задач в реальном времени внутри карточек.
- **🔍 Глобальный поиск**: Быстрый поиск задач по названию во всех проектах.
- **📱 Адаптивный UI**: Современный интерфейс на базе Shadcn/UI, корректно работающий на всех устройствах.
- **🌓 Темная тема**: Поддержка светлого и темного режимов.

---

## 🛠 Технологический стек

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS, Shadcn/UI, Lucide Icons.
- **Backend**: Next.js API Routes (Server Actions), Prisma ORM 7.
- **Database**: PostgreSQL (Neon.tech — Serverless Postgres).
- **State Management**: TanStack Query (React Query) для синхронизации данных с сервером.
- **Drag & Drop**: @hello-pangea/dnd.
- **Validation**: Zod (валидация схем на фронтенде и бэкенде).
